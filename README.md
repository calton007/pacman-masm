# pacman-masm

这是一个用 MASM 风格 16 位 x86 汇编写的吃豆人游戏项目。

项目当前只有一个核心源码文件：[PACMAN.asm](PACMAN.asm)。程序依赖 DOS 环境，使用 BIOS/DOS 中断完成显示、键盘输入、计时器和声音控制，因此不能在现代 64 位 Windows 终端里直接运行。

## 项目内容

| 文件 | 说明 |
|---|---|
| `PACMAN.asm` | 游戏完整源码，包含菜单、地图、玩家移动、幽灵移动、计分、生命值、胜利和失败逻辑 |
| `.gitattributes` | Git 文本换行设置 |
| `.gitignore` | 忽略 MASM/TASM 编译产物、DOSBox 临时文件和本地工具目录 |

## 运行环境

这个项目需要 DOS 兼容环境和 16 位汇编工具链。

推荐环境：

| 工具 | 用途 |
|---|---|
| DOSBox | 运行 DOS 程序 |
| MASM 5.x/6.x 或 TASM | 汇编源码 |
| DOS LINK | 链接生成 DOS 可执行文件 |

现代 Visual Studio 自带的 MASM 主要面向 Windows 程序，不是这个项目的首选工具链。

## 静态检查结果

当前源码存在会影响正常运行的问题：

| 位置 | 问题 | 影响 |
|---|---|---|
| `INITIAL PROC` | 过程末尾缺少 `RET` | 初始化后不会正确返回主流程 |
| 保存 `OLD1CH` 处 | `ES` 被写到了 `OLD1CH` 低 16 位 | 恢复计时器中断时地址错误，可能导致程序退出或结束时异常 |
